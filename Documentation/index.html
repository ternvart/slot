C) Installation - top

Very simple installation:

  • Upload files to your server or hosting
  • Open install.php page and complete steps
  • Don't forget to delete install.php and db.sql files after installation
  • You can access admin area on: yourwebsite.com/admin
  • Website sitemap located at: yourwebsite.com/sitemap.xml

Plugin installation:

  • You can get YouTube plugin here
  • Enter information and download file
  • Upload file to /system/classes folder

If you have problems with installation check chmod values of files /system/storage folder. Make all chmods 775.
